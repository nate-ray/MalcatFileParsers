## File Parsers

 - LHA - LHA (or LZH) is an archive file format, popular in the 1990s and heavily used on Amiga systems for file compression. It uses algorithms similar to ZIP, often denoted by tags like -lz1-. While largely replaced by ZIP or 7z, it remains active in retro computing. Many modern archivers (e.g., 7-Zip, WinRAR) can open, extract, or manage .lha files.
